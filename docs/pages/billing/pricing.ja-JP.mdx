# 料金設定

Zeabur は、プロジェクトに対して消費ベースの料金モデルを提供しています。使用しないリソースに対しては請求されません。

代わりに、Zeabur は、サービスが実際に使用する各 vCPU とメモリリソースの利用時間に応じて料金を請求します。

## 料金レート

Zeabur は、以下の式で使用料金を計算します：

`総サービス時間`（分）x （`メモリ使用量`（GB）+ `vCPU使用量`（コア））

利用料金は以下の通りです：

- メモリ - **0.00023 米ドル** / GB-minute（約 10 米ドル / GB-month）

- vCPU - **0.0005 米ドル** / vCPU-minute（約 20 米ドル / vCPU-month）

- ネットワークの転送 - Serverless プランでは最初の 10GB は無料、Developer プランと Team プランで は 100GB まで無料。その後、**10 米ドル** / 100GB。

最低サービス料金はありませんし、サービスがリソースを使用しない場合やクラッシュした場合には請求されません。

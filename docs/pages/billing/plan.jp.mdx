# プラン

Zeabur では、さまざまなニーズに応じたサブスクリプションプランを提供しています。

機能と価格については、[Zeabur の「価格」ページ](https://zeabur.com/pricing)で確認できます。このドキュメントでは、各プランのターゲットユーザーとサポートされる支払い方法について説明します。

## サーバーレスプラン

デフォルトでは、すべてのZeaburアカウントはサーバーレスプランです。
数の制限なしでサーバーレス関数としてサービスをデプロイすることができます。

<Callout>
このプランではコンテナ化されたサービスもデプロイできますが、このプランにアップグレードしない限り、それらのサービスはいつでも削除される可能性があります。

テストや評価に適しています。
</Callout>

## 開発者プラン

開発者プランは、あらゆる種類の開発者や小規模チームに適しています。豊富なリソースクォータや高いサポート優先度、Discord サーバーでの専用の「Dev Plan」バッジに加えて、制限に悩むことなく Zeabur でさまざまな興味深いアプリケーションを作成し、「ペイアズユーゴー」もできます。

開発者プランでは、サーバーレス関数とコンテナ化されたサービスの両方をデプロイできます。

開発者プランには、毎月5ドルの無料クォータがあります：

- 1か月の使用料が5ドルを超えない場合、このプラン以外の追加料金は請求されません。
- 使用料が5ドルを超える場合、超過分の使用料は["価格"](./pricing.jp.mdx)の式に従って請求されます。

## チームプラン

チームプランは、中規模および大規模なリソース要件を持つプロジェクトに適しています。より多くのリソースクォータを提供するだけでなく、Zeaburチームと直接連絡を取り、さまざまな可能性について議論することもできます。

チームプランでは、サーバーレス関数とコンテナ化されたサービスの両方をデプロイできます。

クレジットカードとAlipayの支払いに対応しています。他の支払い方法が必要な場合は、お問い合わせください。

## エンタープライズプラン

チームプランが要件を満たさない場合は、[お問い合わせ](mailto:contact@zeabur.com)して、使用シナリオに合わせたプランを取得することができます。

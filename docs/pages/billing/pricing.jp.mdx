# 料金設定

Zeabur は、プロジェクトに対して消費ベースの料金モデルを提供しています。使用しないリソースに対しては請求されません。

代わりに、Zeabur は、サービスが実際に使用する各vCPUとメモリリソースの利用時間に応じて料金を請求します。

## 料金レート

Zeabur は、以下の式で使用料金を計算します：

`総サービス時間`（分）x （`メモリ使用量`（GB）+ `vCPU使用量`（コア））

利用料金は以下の通りです：

- メモリ - **US$0.00023** / GB-minute（約US$10 / GB-month）

- vCPU - **US$0.0005** / vCPU-minute（約US$20 / vCPU-month）

- ネットワークの転送 - Serverless プランでは最初の10GBは無料、Developer プランと Team プランで は 100GB まで無料。その後、**US$10** / 100GB。

最低サービス料金はありませんし、サービスがリソースを使用しない場合やクラッシュした場合には請求されません。
